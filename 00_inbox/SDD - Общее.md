## Определение

Спецификация (документация) является истиной для человека и для ИИ.

существует несколько уровней его реализации:
**Сначала составляется спецификация** : сначала пишется тщательно продуманная спецификация, которая затем используется в процессе разработки с помощью ИИ для решения поставленной задачи.
**Спецификация, привязанная к проекту** : Спецификация сохраняется даже после завершения задачи, чтобы продолжать использовать ее для развития и поддержки соответствующей функции.
**Спецификация как исходный код** : Спецификация является основным исходным файлом на протяжении времени, и только спецификация редактируется человеком, человек никогда не трогает код.
