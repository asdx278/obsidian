# Claude - Skills

> [!tldr] ai review
> 

## Что такое skills?

Skills — это папки с инструкциями, скриптами и ресурсами, которые Claude динамически загружает для повышения производительности при выполнении специализированных задач.
Skills помогают claude выполнять задачи в соответствии с определенными требованиями и рекомендациями, повторяемым способом.

При постановке задачи, claude просматривает доступные skills и выбирает релевантные, также предотвращая перегрузку контекстного окна.

Дополняют общие знания claude.
Можно описать лучшие практики, требования к процессу.

Skills описываются в markdown, а также есть возможность добавить скрипты к custom skills.

Skills предоставляют специализированные процедуры, которые активируются динамически при необходимости и работают везде в Claude.

Skills помогают claude лучше использовать MCP.

Инструкции для чатов, skills для задач.

Механизм навыков полностью зависит от того, есть ли у модели доступ к файловой системе, инструментам для навигации по ней и возможность выполнять команды в этой среде.

## Типы

### Anthropic

Это skills, созданные anthropic, такие как создание документов word, excel, powerpoint, pdf. Доступны всем пользователям.

### Custom

Собственные skills, специфичные для выполнения определенных задач.

### Сравнение skills с другими возможностями claude

#### Skills vs. Projects

[[Claude - Projects]] предоставляют статические фоновые знания, которые всегда загружаются при запуске чатов в них. Skills предоставляют специализированные процедуры, которые активируются динамически при необходимости и работают везде в Claude.

#### ## Skills vs. MCP (Model Context Protocol)

MCP подключает Claude к внешним сервисам и источникам данных. Skills предоставляют процедурные знания — инструкции по выполнению конкретных задач или рабочих процессов. Вы можете использовать оба вместе: подключения MCP дают Claude доступ к инструментам, а Skills обучают Claude тому, как эффективно использовать эти инструменты.

#### ## Skills vs. Custom Instructions

[[Claude - Custom instructions]] применяются широко ко всем вашим разговорам. Skills специфичны для задач и загружаются только при необходимости, что делает их лучше для специализированных рабочих процессов.
