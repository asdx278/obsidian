# Claude - Skills

> [!tldr] ai review
> 

## Что такое skills?

Skills — это папки с инструкциями, скриптами и ресурсами, которые Claude динамически загружает для повышения производительности при выполнении специализированных задач.
Skills помогают claude выполнять задачи в соответствии с определенными требованиями и рекомендациями, повторяемым способом.

При постановке задачи, claude просматривает доступные skills и выбирает релевантные, также предотвращая перегрузку контекстного окна.

Дополняют общие знания claude.
Можно описать лучшие практики, требования к процессу.

Skills описываются в markdown, а также есть возможность добавить скрипты к custom skills.

Skills предоставляют специализированные процедуры, которые активируются динамически при необходимости и работают везде в Claude.

Skills помогают claude лучше использовать MCP.

Инструкции для чатов, skills для задач.
## Типы

### Anthropic

Это skills, созданные anthropic, такие как создание документов word, excel, powerpoint, pdf. Доступны всем пользователям.

### Custom

Собственные skills, специфичные для выполнения определенных задач.
